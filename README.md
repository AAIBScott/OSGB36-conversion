# OSGB36-conversion
C# program written in Visual Studio.  Requires a 'button' to start

Based on https://github.com/dstl/osgb/tree/master/src/main/java/uk/gov/dstl/geo/osgb
and https://www.ordnancesurvey.co.uk/documents/resources/guide-coordinate-systems-great-britain.pdf

NOTE does not produce same accuracy as GridInquest
